# TaskFlow User Guide (System Flows)

This guide is for end users and subscribers of the TaskFlow system.
It explains the normal flow of using the platform from first visit to daily operations.

## 1. Who Uses the System

1. Workspace Owner/Admin
2. Employee (Subscriber/User invited to a workspace)

## 2. Entry Flow (First Visit)

1. Open the TaskFlow landing page
2. Click `Get Started` or `Start Free Trial`
3. You will be directed to account creation or login depending on your action

## 3. Owner/Admin Flow

### 3.1 Create Workspace

1. Open the Sign Up page
2. Enter workspace name, full name, and email
3. Complete account creation
4. Log in using your account

### 3.2 Admin Dashboard

1. After login, you will land on the Dashboard
2. Review dashboard cards:
3. Total tasks
4. Completed tasks
5. Ratings and team summaries

### 3.3 Create and Manage Tasks

1. Go to the `Tasks` menu
2. Click `Create Task`
3. Set title, description, due date, and assignees
4. Track progress from task lists and task details

### 3.4 Invite Team Members (Subscribers)

1. Go to the `Invites` menu
2. Send invite to employee email
3. Employee receives invite link
4. Employee opens the invite link and joins the workspace

### 3.5 Billing and Seat Management

1. Go to the `Billing` or `Workspace Billing` section
2. Check plan status and seat usage
3. Update seats if needed (admin only)

## 4. Employee/Subscriber Flow

### 4.1 Join Workspace

1. Open invite link from admin
2. Complete the Join Workspace form
3. Set password and submit
4. Log in to the system

### 4.2 Daily Work Flow

1. Open the Dashboard
2. Use Time Tracker:
3. Click `Clock In` when starting work
4. Keep monitoring/capture active while working
5. Click `Clock Out` when done

### 4.3 Task Execution

1. Open the `Tasks` section (My Tasks)
2. View assigned tasks and status
3. Upload submissions or updates where required
4. Complete subtasks if available

### 4.4 Communication

1. Open the `Messages` section for direct/group chat
2. Check notifications and updates regularly

## 5. Navigation Reminder Behavior

1. If an employee tries to navigate without clocking in:
2. A reminder modal appears once
3. After that, user can continue navigation
4. After logout and next login, reminder can appear once again

## 6. Notifications (Toaster)

1. Success and error messages appear as toast notifications on the right side
2. Example events:
3. Login success/failure
4. Task create/update actions
5. Invite actions

## 7. Account and Security Basics

1. Use `Forgot Password` from the login screen when needed
2. Update personal details in the `Profile` section
3. Always click `Logout` when finished

## 8. Common User Paths (Quick Map)

1. Visitor: Landing Page -> Sign Up -> Login -> Dashboard
2. Admin setup: Dashboard -> Invites -> Tasks -> Billing
3. Employee setup: Invite Link -> Join Workspace -> Login -> My Tasks
4. Daily employee cycle: Dashboard (Clock In) -> My Tasks -> Messages -> Dashboard (Clock Out)

## 9. If Something Fails

1. Refresh the page and retry
2. Check your session by logging out and logging in again
3. Verify invite token is valid (if joining workspace)
4. Contact workspace admin for access, role, or seat-limit issues
